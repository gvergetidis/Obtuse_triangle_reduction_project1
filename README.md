Κατάλογο των αρχείων κώδικα και ο ρόλος τους
Στο κατάλογο Build τοπόθετούνται τα αυτόματα παραγώμενα αρχεία του cmake καθώς και το Makefile και το CMakeLists.txt που πάραξε το προηγούμενο
Στο κατάλογο Out τοπόθετείτε το αρχείο εξόδου output.json
Στο κατάλαγο .git βρίσκονται αρχεία σχετιμα με το git
Το αρχείο CCDT.h ορίζει τη κλάση που χρησιμοποιουμε (η οποια κληρονομέι την CostrainedDelaunyTraingulation της CGAL)


Η είσοδος στο πρόγραμμα είναι ένα αρχέιο json που διαβάζεται με την χρήση της βιβλιοθήκης boost 1.86.
Αφού το πρόγραμμα διαβάσει το αρχέιο json κατασκευάζει μία τριγονοποίηση Delauny.
Χρησημοποιήται η κλάση CCDT που ορίζεται στα παρακάτο αρχεία .........
Αυτή κληρονομεί την CDT της CGAL προσθέτοντας την insert_no_flip() που προσθέτει σημείο ώστε να μην γίνεται flip μετά την εισαγωγή του σημείου. Ακόμα στην κλάση υπάρχουν πολλές βοηθητικές συναρτήσεις όπως ...
Το πρόγραμμα ξεκινάει δοκιμάζοντας να κάνει flip όσες ακμές μπόρει για να μειώσει τις αμβλείες γωνίες της τριγωνοποιησης.
Στην συνέχεια σε αυτή την τριγονοποίηση δοκιμάζονται οι παρακάτω τρόποι εισαγωγής steiner point.
1. Εισαγωγή στο μέσο της απένατνι πλευράς από την αμβλεία
2. Εισαγωγή στο περίκεντρο (ή αν αυτό είναι εκτός του region boundry στο βαρύκεντρο) του αμβληγώνειου τριγώνου
3. Εισαγωγή στο κέντρο του κυρτου πολυγώνου γειτωνικών αμβλειγώνιων τριγώνων
4. Εισαγωγη στη προβολή της αμβλέιας γωνιάς στην απέναντι πλευρά
Κάθε τρόπος εισαγωγής δοκιμάζεται με όριο 20 περάσματα των faces της τριγωνοποιήσης




Οδηγίες μεταγλώττισης
Για δημιουργία Makefile τρέξτε την παρακάτω εντολή:
cmake -S Build -B Build
Για μεταγλώττιση τρέξτε αυτή την εντολή:
make -C Build

Οδηγίες χρήσης του προγράμματος
Το πρόγραμμα πέρνει ως είσοδο στην γραμμή εντολών το όνομα του αρχέιου με την είσοδο.
Ένα παράδειγμα χρήσης είναι ./ex nameoffile.json
